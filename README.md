# C23-PS117 | Cloud Computing 
### Install

    npm install

### Run the app

    npm start

## API Documentation

For more details, please check `/api-docs`

### Authentication

`GET /users/`

    Get all users

`POST /register/`

    Register a new user

`POST /login/`

    User login

`POST /update-bmr/`

    Update user BMR

### Users

`GET /users/{id}`

    Get user data

`POST /users/{id}`

    Add new user data

`PUT /users/{id}`

    Update user data
